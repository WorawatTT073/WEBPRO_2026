สูตรกะเพรา

ขั้นตอนการทำกะเพรา

1. หั่นหมูสันคอออกเป็นชิ้นๆ แล้วสับหมูออกให้ละเอียดตามความต้องการ แล้วพักไว้
2. โขลกพริกแห้งให้ละเอียด แล้วใส่พริกแดงจินดา และกระเทียมโขลกตามลงไปให้ละเอียด (สามารถปั่นแทนได้ ตามความชอบ)
3. ตั้งกระทะให้ร้อน ใส่น้ำมันพืชลงไปพอประมาณ ผัดพริกและกระเทียมให้มีกลิ่นหอม
4. เมื่อพริกกระเทียมหอมได้ที่แล้ว ใส่หมูสับลงไปผัดให้กระจาย ไม่เกาะกันเป็นก้อน ผัดให้เข้ากับพริกกระเทียม ให้หมูสุกในระดับนึง แล้วปรุงรสด้วย น้ำมันหอย น้ำปลา ซอสปรุงรส และซีอิ๊วดำหวาน น้ำตาล เล็กน้อย เร่งไฟผัดให้เข้ากัน ชิมรสชาติ แล้วใส่ใบกะเพราลงไปผัดให้เข้ากัน ปิดเตา ตักเสิร์ฟ

สูตรแกงส้มหน่อไม้ดอง

ขั้นตอนการทำแกงส้มหน่อไม้ดอง

1. นำหน่อไม้ดองไปต้มแล้วเทน้ำทิ้งสัก 1 รอบ ตักขึ้นพักให้สะเด็ดน้ำ
2. นำน้ำพริกแกงส้มไปละลายน้ำ นำขึ้นตั้งไฟให้เดือด ปรุงรสด้วยน้ำมะขามเปียก น้ำปลา และน้ำตาลปี๊บตามชอบ
3. นำน้ำพริกแกงส้มไปละลายน้ำพอน้ำแกงเดือดจัดใส่ปลาลงไปต้มให้สุก ระหว่างนี้ห้ามคนเด็ดขาดเพื่อไม่ให้คาว (กรณีใช้กุ้งสดให้ข้ามขั้นตอนนี้ไป โดยใส่กุ้งสดในขั้นตอนสุดท้าย)
4. นำน้ำพริกแกงส้มไปละลายน้ำใส่หน่อไม้ที่เตรียมไว้ลงไป พอเดือดชิมรสอีกครั้งให้ถูกใจ
5. นำน้ำพริกแกงส้มไปละลายน้ำใส่ใบมะกรูดแล้วก็ปิดไฟค่ะ (หากใช้ปลาทะเลที่มีกลิ่นคาวจัด ควรเพิ่มน้ำมะกรูด และลูกมะกรูดลงไปช่วยดับคาว)
6. นำน้ำพริกแกงส้มไปละลายน้ำตักใส่ชาม พร้อมเสิร์ฟ

สูตรส้มตำปูปลาร้า

ขั้นตอนการทำตำปูปลาร้า

1. โขลกพริกกับกระเทียมพอหยาบ
2. ใส่มะเขือเทศ ปูดอง ปรุงด้วยน้ำปลาร้า น้ำมะนาว และน้ำตาลปี๊บ
3. ใส่มะละกอ ขนมจีน คลุกเคล้าให้เข้ากัน
4. ชิมรสตามชอบ พร้อมเสิร์ฟ

สูตรแกงเขียวหวาน

ขั้นตอนการทำแกงเขียวหวาน

1. ผัดพริกแกงเขียวหวานกับหัวกะทิให้หอม
2. ใส่เนื้อไก่ ผัดจนสุก
3. เติมกะทิ ปรุงรสด้วยน้ำปลา น้ำตาลปี๊บ
4. ใส่มะเขือ ใบโหระพา ต้มจนเดือด

สูตรพริกแกง

ขั้นตอนการทำพริกแกง

1. ใส่พริกแห้ง หอมแดง กระเทียม ข่า ตะไคร้ ลงครก
2. โขลกให้ละเอียดเข้ากัน
3. ปรุงด้วยเกลือและกะปิ
4. ใช้เป็นเครื่องแกงตามต้องการ

สูตรไข่เจียว

ขั้นตอนการทำไข่เจียว

1. ตอกไข่ใส่ชาม ปรุงด้วยน้ำปลา
2. ตีให้เข้ากันจนฟู
3. ตั้งน้ำมันให้ร้อน เทไข่ลงทอด
4. ทอดจนเหลืองกรอบ ตักขึ้นสะเด็ดน้ำมัน

สูตรไข่ดาว

ขั้นตอนการทำไข่ดาว

1. ตั้งกระทะใส่น้ำมันให้ร้อน
2. ตอกไข่ลงกระทะเบา ๆ
3. ทอดจนไข่ขาวสุก ไข่แดงตามระดับที่ชอบ
4. ตักขึ้นพร้อมเสิร์ฟ

อ้างอิง

สูตรมาม่าผัด

ขั้นตอนการทำมาม่าผัด

1. ตั้งกระทะไฟแรงตามด้วยน้ำเปล่า รอน้ำเดือดจัด นำบะหมี่กึ่งสำเร็จรูปลงไปต้มจนสุกตักขึ้นพักไว้
2. ตั้งกระทะไฟกลางตามด้วยน้ำมันถั่วเหลือง นำพริกขี้หนูและกระเทียมปั่นละเอียดผสมน้ำลงไปผัด ผัดจนหอมตามด้วยหมูสับ แครอท และข้าวโพดอ่อน ผักจนสุก
3.ตามด้วยหมูยอทอด ผัดให้เข้ากัน ปรุงรสด้วย น้ำตาลทราย ผงปรุงรส และซอสหอยนางรม

สูตรยำมาม่าทะเล

ขั้นตอนการทำยำมาม่าทะเล

1. ลวกมาม่า กุ้งสด ปลาหมึก ปูอัด และหอยแมลงภู่ เตรียมไว้<
2. ทำน้ำยำโดยผสมพริกขี้หนู น้ำตาลทราย น้ำปลา และน้ำมะนาว คนจนน้ำตาลละลาย
3. ส่เส้นมาม่า และเครื่องทะเลต่าง ๆ ลงไปคลุกกับน้ำยำจนเข้ากัน ใส่หอมใหญ่ มะเขือเทศ และขึ้นฉ่าย เคล้าจนเข้ากันอีกครั้ง

สูตรหมูทอดกระเทียมพริกไทย

ขั้นตอนการทำหมูทอดกระเทียมพริกไทย

1. นำเนื้อหมูหั่นใส่ลงในชามผสม ปรุงรสด้วยสามเกลอพร้อมใช้ ตราคนอร์ ซอสผัดเข้มข้น ตราคนอร์ ผงปรุงครบรส รสหมู ตราคนอร์ อร่อยชัวร์ พริกไทยดำบดและน้ำมันพืช คลุกเคล้าให้เข้ากัน หมักทิ้งไว้ประมาณ 1 ชั่วโมง
2. ใส่แป้งทอดกรอบ ตราคนอร์และน้ำสะอาดลงไป คลุกเคล้าให้เข้ากัน
3. ตั้งกระทะน้ำมันบนไฟปานกลาง ใส่หมูลงไปทอด กลับด้านทอดให้เหลืองทั้งสองด้าน
4. ตักขึ้นให้สะเด็ดน้ำมัน นำกระเทียมลงไปเจียวต่อจนเหลืองสวย ตักขึ้นพักไว้สำหรับโรยหน้า
5. จัดเสิร์ฟหมูทอดโรยกระเทียม เสิร์ฟพร้อมข้าวเหนียวร้อนๆ
